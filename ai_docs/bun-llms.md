# Bun Documentation Content

This is a comprehensive documentation index for **Bun**, an all-in-one toolkit for developing modern JavaScript/TypeScript applications.

## Main Categories

### Core Documentation
- **Welcome & Installation**: Getting started with Bun
- **Quickstart**: Build your first app with Bun
- **TypeScript**: Full TypeScript support with type definitions

### Bundler
- Fast native bundler for JavaScript, TypeScript, JSX
- CSS support with modern features
- Bytecode caching for faster execution
- Single-file executable generation
- Fullstack dev server with hot reloading
- HTML & static site building
- Built-in loaders, macros, and plugins
- JavaScript/TypeScript minifier

### Runtime & APIs
- **HTTP Server**: `Bun.serve()` with TLS, routing, WebSockets
- **File I/O**: Read/write files with optimized APIs
- **Networking**: Fetch, TCP, UDP, DNS
- **Databases**: Native bindings for SQL, SQLite, PostgreSQL, MySQL
- **Redis**: Native Promise-based client
- **S3**: Native S3-compatible object storage bindings
- **FFI**: Call native libraries from JavaScript
- **Workers**: Multi-threaded JavaScript execution
- **Shell**: Run shell commands from JavaScript
- **Child Processes**: Spawn and manage subprocesses
- **Streams**: Work with binary data efficiently
- **Hashing & Cryptography**: Password hashing, various crypto algorithms
- **Archive**: Create and extract tar archives

### Package Manager
- `bun install`: Fast package installation
- `bun add/remove`: Dependency management
- `bun link`: Local package linking
- `bun publish`: Publish to npm registry
- Monorepo support with workspaces
- Lockfile management
- Private registry configuration
- Lifecycle scripts

### Testing
- Built-in Jest-compatible test runner
- Code coverage reporting
- Snapshot testing
- Mocking and spying
- Watch mode
- DOM testing with happy-dom
- Lifecycle hooks

### Guides (Extensive Coverage)
- **Binary Data**: Convert between ArrayBuffer, Blob, Buffer, Uint8Array, DataView, strings
- **Streams**: Convert Node.js Readable and ReadableStream formats
- **File Operations**: Read/write files in various formats (JSON, Buffer, ArrayBuffer, etc.)
- **Process**: Arguments, signals, IPC, stdin/stdout/stderr
- **HTTP**: Server creation, fetch, file uploads, proxying, TLS, clustering
- **WebSockets**: Simple servers, pub/sub, compression, context data
- **Deployment**: AWS Lambda, DigitalOcean, Google Cloud Run, Railway, Render, Vercel
- **Ecosystem Integration**: Astro, Discord.js, Docker, Drizzle ORM, Elysia, Express, Hono, Mongoose, Next.js, Nuxt, PM2, Prisma, Qwik, React, Remix, Sentry, SolidStart, SvelteKit, TanStack Start, Vite
- **Utilities**: Base64, deep equality, DEFLATE/gzip compression, UUID generation, timezone handling, version management

### Additional Features
- **Debugging**: VS Code extension, web debugger
- **JSX**: Built-in support with configurable transpilation
- **Module Resolution**: Import resolution and handling
- **Globals**: Bun-specific global objects
- **Auto-install**: Automatic package installation for standalone scripts
- **Plugins**: Universal plugin API for runtime and bundler extension
- **Secrets API**: Secure credential storage
- **Color**: Format colors as CSS, ANSI, hex
- **YAML/TOML**: Native file format support
- **Environment Variables**: .env file support
- **Watch Mode**: --watch and --hot modes
- **Web APIs**: Server-side Web Standard APIs support
